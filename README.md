# Academic Paper Network Analysis System

学术论文网络分析系统 - 基于图论算法的论文关联分析

## 项目简介

本项目实现了一个完整的学术论文网络分析系统，通过提取论文关键词构建论文关系图，并使用经典图论算法进行网络分析。系统能够揭示论文之间的潜在关联，识别研究热点，并计算论文之间的相似度和距离。

## 核心功能

### 1. 数据预处理

**论文数据解析：**
- 从 CSV 文件读取学术论文元数据
- 解析论文字段：ID、标题、年份、引用次数、关键词等
- 处理包含特殊字符和引号的复杂字段
- 提取并解析关键词列表

**关键词分析：**
- 统计所有关键词的出现频率
- 使用快速排序（QuickSort）对关键词按频率降序排列
- 识别 Top 3 高频关键词
- 基于 Top 3 关键词筛选相关论文

### 2. 论文关系图构建

**图结构设计：**
- **节点（Vertex）：** 每篇论文作为一个节点
- **边（Edge）：** 两篇论文之间的共同关键词数量
- **权重（Weight）：** 共同关键词的数量（表示论文相似度）
- **图类型：** 无向加权图

**构建策略：**
- 从筛选后的论文中提取前 1000 篇构建图
- 计算任意两篇论文的共同关键词数量
- 仅当存在共同关键词时才建立边连接
- 使用邻接表（Adjacency List）存储图结构

### 3. 图遍历算法

#### BFS（广度优先搜索）

**功能：**
- 层次遍历论文网络
- 计算节点层级（距离起始论文的跳数）
- 记录前驱节点，构建遍历树
- 统计各层节点数量

**应用场景：**
- 发现与某篇论文最相关的"近邻"论文
- 分析论文影响力的传播层次
- 识别研究社区的边界

#### DFS（深度优先搜索）

**功能：**
- 深度优先遍历论文网络
- 记录节点的发现时间和完成时间
- 检测连通分量（研究社群）
- 输出遍历序列

**应用场景：**
- 探索研究主题的深度关联
- 识别独立的研究社群
- 检测图的连通性

### 4. 最小生成树算法

#### Prim 算法

**功能：**
- 构建最小生成树（MST）
- 使用优先队列（最小堆）优化性能
- 计算 MST 的总权重
- 输出 MST 的边列表

**应用场景：**
- 提取论文网络的核心骨架
- 识别最重要的论文关联路径
- 最小化冗余连接，保留关键联系

**输出：**
- MST 边数、总权重、包含节点数
- 保存到 `prim_mst.txt` 文件

### 5. 最短路径算法

#### Dijkstra 算法

**功能：**
- 计算单源最短路径（从一篇论文到所有其他论文）
- 使用优先队列优化时间复杂度
- 记录每个节点的最短距离
- 统计可达节点和平均距离

**应用场景：**
- 寻找论文之间的最短关联路径
- 评估论文在网络中的中心性
- 发现研究主题的转换路径

**输出：**
- 可达节点数、平均距离统计
- 保存到 `dijkstra_shortest_paths.txt` 文件

## 算法实现细节

### 快速排序（QuickSort）
- 时间复杂度：O(n log n) 平均情况
- 用于关键词频率排序
- 降序排列，优先显示高频关键词

### 图算法复杂度

| 算法 | 时间复杂度 | 空间复杂度 | 数据结构 |
|------|-----------|-----------|---------|
| BFS | O(V + E) | O(V) | 队列 Queue |
| DFS | O(V + E) | O(V) | 栈 Stack（递归） |
| Prim | O((V + E) log V) | O(V) | 优先队列 Priority Queue |
| Dijkstra | O((V + E) log V) | O(V) | 优先队列 Priority Queue |

其中 V = 节点数（论文数），E = 边数（论文关联数）

## 使用方法

### 编译运行

```bash
g++ experiment_4_graph.cpp -o graph
./graph
```

请输入起始节点 ID (0-999): 0

## 输入输出

### 输入数据格式
- CSV 格式的学术论文数据
- 必须包含字段：ID, Title, Year, Cited, Keywords
- Keywords 格式：`[keyword1, keyword2, keyword3, ...]`

### 输出文件

1. **graph_nodes.txt**
   - 所有节点的详细信息
   - 格式：`节点ID 论文ID 标题`

2. **adjacency_list.txt**
   - 图的邻接表表示
   - 格式：`节点ID: 邻居1(权重1) 邻居2(权重2) ...`

3. **prim_mst.txt**
   - Prim 算法的最小生成树
   - 包含总权重和边列表

4. **dijkstra_shortest_paths.txt**
   - Dijkstra 算法的最短路径
   - 包含所有可达节点的距离

## 技术栈

- **编程语言：** C++11
- **数据结构：**
  - 邻接表（Adjacency List）- vector<vector<pair>>
  - 优先队列（Priority Queue）- 最小堆
  - 哈希表（Unordered Map）- 关键词统计
  - 集合（Set）- 快速查找
  - 队列（Queue）- BFS
  - 栈（Stack）- DFS 递归
- **标准库：**
  - `<vector>`, `<queue>`, `<stack>` - 容器
  - `<unordered_map>`, `<set>` - 关联容器
  - `<algorithm>` - 排序算法
  - `<limits>` - 数值极限

## 算法特点

### 优势
- **高效性：** 使用优先队列优化 Prim 和 Dijkstra 算法
- **可扩展性：** 支持大规模论文网络（1000+ 节点）
- **灵活性：** 用户可自定义起始节点
- **完整性：** 覆盖图的遍历、生成树、最短路径等核心算法

### 创新点
- 将论文数据映射为图结构
- 使用共同关键词数量作为相似度度量
- 结合多种图算法进行综合分析
- 自动生成可视化友好的输出文件
